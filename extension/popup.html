Interview Assistant

Resume, JD, and prompt templates are saved in this profile. When live.py is running, non-empty fields sync to the bridge on open, while typing, and on blur. Empty fields clear the bridge so prep waits until you paste.

Use {resume_text} where the resume text should be inserted.

Use {jd_text} where the job description should be inserted.

Optional placeholders: {resume_text}, {jd_text}.

Use {cleaned_interviewer_intent} where the cleaned caption should be inserted.
